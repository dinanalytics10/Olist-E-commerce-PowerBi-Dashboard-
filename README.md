# Olist-E-commerce-PowerBi-Dashboard-
Sales &amp; Business Performance Dashboard built with Power BI and SQL on the Olist Brazilian E-Commerce dataset (96K+ orders, 2016-2018)
![Dashboard Preview](Dashboard/page1_overview.png)

## Overview
A 4-page interactive Power BI dashboard analyzing 
96,478 delivered orders from the Olist Brazilian 
E-Commerce marketplace between September 2016 and 
October 2018.

## Dashboard Pages
- **Page 1 — Overview:** Revenue, orders, customers, AOV
- **Page 2 — Operations:** Reviews, delivery, payments
- **Page 3 — Sellers:** Top sellers, revenue concentration
- **Page 4 — Products:** Categories, pricing, cancellations

## Key Insights
- $15.42M total revenue
- 92.1% on-time delivery rate
- Late delivery drops reviews by 2.59 points
- Top 10% of sellers = 67% of revenue
- Health & Beauty is the #1 revenue category

## Tools Used
- Power BI (DAX, Power Query)
- SQL (SQLite)
- Dataset: Kaggle — Olist Brazilian E-Commerce
